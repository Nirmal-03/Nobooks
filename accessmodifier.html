Access Modifier

An access modifier, in the context of programming languages like Java, is a keyword that determines the accessibility or visibility of classes, methods, and variables within a program. It specifies the level of access or restrictions placed on these elements and defines who can access them and from where.

There are Four types of Access Modifier


1.Public
 Accessible in all the packages

2.Proctected:
 Accessible to all classes in its own package and only in child class of another package.

3.Private:
 Only with in the class we can access

4.Deafult:
 Only with in the package we can access
