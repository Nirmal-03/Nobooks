Strings


  • In Java, string is an object that represents a sequence of characters.
  • The java.lang.String class is used to create a string object.
  • An array of characters works same as Java string.String is immutable object .

For Example:


char[] ch={'n','o','b','o','o','k','s'};
String s=new String(ch);
Is same as:
String s="nobooks";


Creating a string


There are two ways to create a string:


  • By string Literal
  • By new keyword

String literal:


  • Java String literal is created by using double quotes.
  • The string literal will be placed in the string constant pool.

Example:


String s="welcome";
String s1="Welcome";
String s2="Welcome"; //It doesn't create a new in stance



By new keyword


  • JVM will create a new string object in normal (non-pool) heap memory.
  • The variable s will refer to the object in a heap (non-pool)
