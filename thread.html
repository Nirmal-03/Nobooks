Thread

Threads in Java allow concurrent execution of multiple tasks within a single program. They represent lightweight units of execution that can run independently, allowing for parallelism and improved performance. Threads are used to perform tasks concurrently, handle asynchronous operations, and enhance responsiveness in applications.


How to create a Thread?


There are two ways to create a a thread:
 1. By extending Thread class
2. By implementing Runnable interface.
